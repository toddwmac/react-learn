# How to Use Your React App

## Start the App

1. **Open your browser** to: `http://localhost:3000`

You'll see the Welcome page.

## Navigate to Your Counter App

1. Click the link or type: `http://localhost:3000/counter` in your browser
2. You'll see your first React app with the counter!

## What You're Looking At

Your Counter Page shows:
- **A title** - "My First React App"
- **A counter** - shows "Click count: X"
- **A button** - "Add 1" that you can click
- **A box** - explains React concepts in VB terms

## Try It Out

1. Click the "Add 1" button several times
2. Watch the count increase
3. Each click:
   - React detects the click event
   - Calls the `handleClick()` function
   - Updates the `count` state
   - React automatically re-renders the UI
